Gnome Commander is being developed in a stable and a testing branch, where the test releases are for testing of new ideas. Successful ideas will then get backported to the stable branch.

The GNOME Commander sourcecode

All current and historical releases can be found on the Gnome FTP server. Links to the latest releases of the stable and development versions can be found on this page.

Snapshots

For testing purposes only. The main developer Piotr E. releases development snapshots of both the stable and testing branch of Gnome Commander, these tarballs can be downloaded below.

These may be unstable and are for testing purposes only, do not handle important data with these releases.

The latest snapshot packages:

  • Snapshot of the stable branch
  • Snapshot of the testing branch

To check out the 1.3 testing branch sources from SVN, following command can be used:

Source from SVN

For those of you hwo feel really adventurous, or maybe want to hack on the latest code, can download the latest development sourcecode or individual files directly from SVN (just remember to run "autogen.sh" when compiling). But be ware that there can be serious bugs in code from SVN, this code is for hacking, development and testing purposes.

Gnome Commander has its source tree hosted at gnome.org. In the source tree there is also a detailed changelog.To check out the latest code from SVN, for the stable 1.2 branch, following command below can be used. See our development page regarding SVN sources for the 1.3 testing branch.

svn checkout svn://svn.gnome.org/svn/gnome-commander/trunk

Visit the page in following link for more detailed information about SVN at svn.gnome.org.
