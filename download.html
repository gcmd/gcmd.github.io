Versions

Gnome Commander is going to be developed in a stable and a testing branch, where the test releases are for testing of new ideas. Successful ideas will then get backported to the stable branch.

Source from CVS

For those of you hwo feel really adventurous, or maybe want to hack on the latest code, there is also the possibility to download the sourcecode or individual files directly from CVS. Gnome Commander has its source tree hosted at gnome.org. There is also a detailed changelog. To check out the latest code from CVS folowing command can be used:

cvs -z3 -d:pserver:anonymous@anoncvs.gnome.org:/cvs/gnome co gnome-commander

Visit the page in following link for more detailed information about CVS at cvs.gnome.org.
