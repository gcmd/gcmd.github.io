Mime types

The editing of prefered programs in Gnome Commander is currently boken (v 1.1.7) since Gnome has changed this to follow the freedesktop.org standard of handling this. There are two ways of managing this on user basis; use nautilus or "manually" edit the configuration files that handles this in you home directory.

The Nautilus way:

To use Nautilus for doing this can feel a little awkward for many GCMD users, we usually do not use Gnome Commander because we like usin Nautilus.., and for users using another WM than Gnome Nautilus also kidnaps the users desktop (background and desktop icons) if it not is stared the right way: nautilus --no-desktop. Anyway, here is how to do it:

  • Start Nautilus
  • Mark a file of the mimetype you want to change.
  • Right click and chose "Properties" and then the tab "Open with" .
  • In the "Open with" tab chose "Add" and you will be presented a list with programs, in this dialog you can also click "Use a custom command", click "Add" when you are done and your chosen program will be listed in the "Open with" list where you now can chose it.

The manual way:

Insted of using Nautilus you can do this manually by editing the configuration files that controls what programs that is available in various menus for different mimetypes. These files usually lives in ~/.local/share/applications. There are three types of files here:

  • XXX-usercreated.desktop Where "XXX" stands for the application in question. These files controls which mimetypes the application in question can handle, what command to use for starting it, if it shall run in terminal and so on. 
    [Desktop Entry]
Encoding=UTF-8
Name=display
MimeType=image/png;image/jpeg;
Exec='/usr/bin/display'
Type=Application
Terminal=false
NoDisplay=true
  • mimeinfo.cache This file contains a list of mimetypes with the above files specified for which programs to handle the mimetype in question. There can be more than one XXX-usercreated.desktop specified for every mimetype (semicolon separated). 
    [MIME Cache]
audio/mpeg=xmms-usercreated.desktop;mpg123-usercreated.desktop;
image/png=display-usercreated.desktop
  • defaults.list This file controls which application that is the defalt one, the one that starts when you double click a file. It looks like this:
    [Default Applications]
audio/mpeg=xmms-usercreated.desktop
image/png=display-usercreated.desktop

Please refer to freedesktop.org or your distribution of choise for further details.
