GNOME Commander is a fast and powerful graphical filemanager for the Gnome desktop environment, it has a "two-pane" interface in the tradition of Norton and Midnight Commander. Take a look at the screenshots, images tell a lot.

Features

  • Drag'n drop.
  • Gnome mimetypes.
  • FTP using GnomeVFS ftp module.
  • Rightclick menu with the usual file operations to open, open with.., delete or rename files and folders, and also to set properties like ownership and permissions.
  • Rightclick context menu that easily can be extended by entries calling external applications like viewers, editiors or shell scripts to work on selected files or directories.
  • Quick device access buttons with automatic mounting and unmounting ("no-mount" option for supermount users).
  • Latest accessed folder history.
  • Folder bookmarks.
  • Plugin support under development.
  • Fast fileviewer for text and images.

Gnome Commander is Open Source and released under the GPL licence, and has it's developer site at Savannah.

Latest news 

1.1.7 Released - Feb 12 2006

Gnome Commander 1.1.7 is now released, available at the gnome.org FTP-server. This release has some bugfixes and a couple of new features, this release requires GTK+ of version 2.6.0 or above.

New Features:
  • Superfast internal file viewer for text and images.
  • Copying selected filename(s) to clipboard.
  • Inserting selected file full path to the cmdline when pressing CTRL+SHIFT+ENTER.
  • Some new or updated translations.
  • A couple of new key bindings.

Read more in the release changelog about bugfixes and new features.

New project maintainer! - Feb 06 2006

This project has been unmaintained for quite some time, two years since last release, but it is not unmaintained anymore. Piotr Eljasiak has agreed to maintain and keep on developing this project. One more developer, Assaf Gordon has also joined the project and recently submitted some new code and patches. So, this project is alive again and the work for next release 1.1.7 has begun = more news to come...
